# HLU Tool

HLU tool for ArcGIS Pro

Copyright © 2025-2026 Andy Foy Consulting

## Overview

The HLU tool provides a user-friendly interface for searching, displaying and updating habitat and land use attributes held in a relational database alongside spatial features in ArcGIS Pro. Key benefits include: validation of attribute compatibility; comprehensive history log for every change; simple query interface (e.g. all priority habitat features); reduced data volumes through relational data storage; data extraction in bespoke formats; and improved data retrieval and update performance for high-volume, complex data.

## Requirements

The HLU tool requires the following:

* ArcGIS Pro 3.4 or later (version 3.\* only)
* Microsoft .NET 8 runtime or later
* Microsoft SQL Server 2012 or later OR PostgreSQL 12+, OR Oracle DB 12c R2 or later, OR any ODBC-compatible relational database
